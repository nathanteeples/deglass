# Liquid Glass for degoog

A light, Apple-inspired theme for degoog: native SF Pro system typography, Lucide-style inline icons, rounded translucent surfaces, and a dark mode designed as a deep glass layer rather than an inverted light theme.

## Install

Copy the `liquid-glass-theme` folder into degoog's `data/themes/` directory (or the directory set by `DEGOOG_THEMES_DIR`). Then select **Liquid Glass** in **Settings → Themes**.

The theme deliberately uses the platform's installed SF Pro font through the standard Apple system font stack. It does not ship Apple font files.

## Customize

Use the theme card's **Configure** action to select a blue, violet, or mint accent. The theme also respects degoog's light, dark, and system appearance choices.
