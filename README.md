# deglass

An extension-store repository for [degoog](https://github.com/degoog-org/degoog).

## Available themes

### Liquid Glass

An Apple-inspired theme with restrained translucent surfaces, SF Pro system typography, and light/dark modes. It preserves Degoog's original layouts and icons, with blue, violet, and mint accent choices from the theme configuration panel.

## Add to degoog

1. Open **Settings → Store** in degoog.
2. Select **Add**.
3. Paste the repository clone URL:

   `https://github.com/nathanteeples/deglass.git`

4. Refresh the source, then install **Liquid Glass** from the Themes section.

This repository follows the [Degoog Store repository format](https://degoog-org.github.io/docs/store.html).
